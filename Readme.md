Проект HedgehogCheckLib містить код для знаходження мінімальної кількості зустрічеї їжаків для зміни їх усіх у один колір.
Проект HedgehogCheckLibTest тестує правильність роботи алгоритму.

В папці Sudoku є файли sudoku_test.js і sudoku_test.test.js.
Перший надає алгоритм перевірки рішення судоку на валідність, у другому написані тести для перевірки роботи.
